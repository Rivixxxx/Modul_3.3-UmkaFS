1. Установка SIM-карты

  1. Открутите винты корпуса терминала с помощью крестовой отвертки PH1.
  2. Осторожно снимите крышку и извлеките плату.
  3. Установите SIM-карту в соответствующий слот.
  4. Соберите корпус обратно.
Внимание: Убедитесь, что питание терминала отключено перед установкой SIM-карты!

2. Подключение питания

  1. Подключите терминал к источнику питания 12В.
  2. Используйте разъем: 1 (красный +) и 7 (черный -) пины для подачи питания.
  3. Убедитесь, что все модули терминала запустились.
Информация: Питание по USB или от встроенной батареи недостаточно для работы всех модулей.

3. Подключение датчиков

  • Датчик зажигания
  • Проводные ДУТ (RS-485)
  • Беспроводные ДУТ (BLE)

3.1 Подключение датчика зажигания

  1. Синий провод (дискретный вход-выход) подключите к плюсовому проводу после замка зажигания.

3.2 Подключение проводных ДУТ (RS-485)

  1. Подключите до 3 датчиков уровня топлива (ДУТ) с протоколом LLS по интерфейсу RS-485.
  2. Используйте кабель типа «витая пара».
  3. Резистор на конце шины — 120 Ом.
  4. Минимизируйте длину ответвлений от шины к датчикам.
  5. Назначьте каждому устройству уникальный адрес для предотвращения коллизий.
Информация: Максимум 3 проводных ДУТ на одну шину RS-485.

3.3 Подключение беспроводных ДУТ (BLE)

  1. В конфигураторе перейдите во вкладку Система → Параметры Bluetooth.
  2. Выберите режим BLE: ДУТы BLE (BLEMODE 2) или Конфигурирование и ДУТы BLE (BLEMODE 3).
  3. Запишите конфигурацию в терминал.
  4. На вкладке ДУТы BLE введите MAC-адрес или используйте BLE-сканер для поиска устройств.
  5. Поставьте галочку «Опрашивать» для начала получения данных.
  6. Адресация беспроводных ДУТ начинается с 7.
Успех: Можно подключить до 4 беспроводных ДУТ одновременно.

4. Установка и запуск конфигуратора

  1. Скачайте установщик ПО «Конфигуратор 3XX» с официального сайта.
  2. Запустите скачанный файл и разрешите внесение изменений.
  3. Выберите язык и путь установки.
  4. При первой установке выберите опцию Установить драйвер терминала.
  5. Дождитесь завершения установки и запустите конфигуратор.
Информация: Для работы требуется Windows 7 или выше.

5. Подключение терминала к ПК

  1. Запустите приложение «Конфигуратор УМКа3ХХ» через меню «Пуск».
  2. Подключите терминал к ПК с помощью USB A - mini-B кабеля или через BLE.
  3. Для BLE: нажмите «Удаленное конфигурирование», введите IMEI и пароль, нажмите «Подключиться».
Внимание: При первом запуске программа может предложить обновление — рекомендуется согласиться.

6. Настройка сервера

  1. Перейдите во вкладку Серверы в конфигураторе.
  2. Укажите IP-адрес или домен и порт сервера мониторинга.
  3. Запишите конфигурацию на терминал с помощью соответствующей кнопки.

7. Установка терминала на транспортное средство

  1. Установите терминал так, чтобы антенна ГЛОНАСС/GPS была направлена вверх.
  2. Расположите терминал на открытом месте, вдали от металлических предметов.
  3. Используйте пластиковые стяжки или специальные разъемы для крепления проводов.
  4. Дайте терминалу несколько минут для связи с сервером обновлений.
Внимание: Соблюдайте ориентацию устройства для корректной работы навигации!

8. Заведение объекта на сервере и привязка IMEI

  1. Войдите в систему мониторинга ГЛОНАССSoft.
  2. Перейдите в раздел Объекты и нажмите на иконку добавления нового объекта.
  3. Заполните карточку объекта:
    • Имя объекта (гос.номер)
    • IMEI терминала
    • Тип устройства
    • Модель объекта
    • Подразделение (по необходимости)
    • Телефоны SIM1 и SIM2 (по необходимости)
Успех: После выполнения всех шагов терминал будет готов к работе!
